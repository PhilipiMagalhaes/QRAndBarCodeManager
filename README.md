# QRAndBarCodeManager
 ## Simple app for scanning and creating QR codes and bar codes.

### Packages used:
- ZXing.Net.Mobile
- Xam.Plugins.Vibrate
- sameerIOTApps.Plugin.SecureStorage
- Plugin.Permissions
- Xamarin.Essentials
